FCM JBrowser8 설치 ​

파일 다운로드 ​

브라우저 상단 다운로드 URL 확인: https://fcm.hyundai-ifc.com/JbrowserSetup.html

다운로드 URL 페이지 접속 후 J.Browser8 설치 클릭

주의사항: J.Browser 설치 오류시 .Net Framework 4.5.2 우선 설치 필요

설치 절차 ​

1. 설치 파일 실행 ​

다운로드 받은 JBrowser8Setup.exe 실행합니다.

2. 언어 선택 ​

언어 선택 후 OK 버튼 클릭

3. 설치 시작 ​

설치 시작 창에서 다음 버튼 클릭

4. 사용권 계약 ​

위 사항에 동의합니다 선택 후 다음 버튼 클릭

5. 설치 경로 확인 ​

설치 PATH 확인: 설치위치 변경 불가

설치 버튼 클릭

6. 설치 완료 ​

설치가 완료되면 다음 단계로 진행합니다.

실행 방법 ​

시작 > JBrowser8 폴더 클릭 > JBrowser8 실행
